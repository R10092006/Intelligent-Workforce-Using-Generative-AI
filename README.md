# Intelligent Workforce Management System using AI

## 📌 Problem Statement
Managing workforce tasks manually is inefficient and lacks prioritization.

## 💡 Solution
This system uses AI-based logic to analyze tasks and categorize them into:
- High Priority Tasks
- Meeting Scheduling
- Risk Detection
- Normal Tasks

## 🚀 Features
- Task input system
- AI-based task analysis
- Priority classification with color indicators
- Task history tracking
- Database storage using SQLite
- Clean and interactive UI
